# Atopos Security Documentation Notice

This repository contains security documentation and publish-safe evidence artifacts for Atopos.

## Official sources
Official security documentation is published only via:
- https://atoposlabs.com
- https://github.com/wicki2c/atopos-security

## Modification and attribution
This repository is licensed under CC BY 4.0. Modified versions must not be represented as official Atopos documentation.

## Evidence artifacts
Evidence artifacts should not be altered in a way that changes their meaning or misrepresents completeness/status.

## Responsible disclosure
If you discover a vulnerability, please follow [DISCLOSURE_POLICY.md](./DISCLOSURE_POLICY.md).
