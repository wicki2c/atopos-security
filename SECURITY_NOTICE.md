# Atopos Security Documentation Notice

This repository contains security documentation, threat models,
claims matrices, and publish-safe evidence artifacts for Atopos.

The materials are licensed under CC BY 4.0. However:

1. Modified versions must not be represented as official Atopos documentation.
2. Evidence artifacts must not be altered in a way that changes their meaning.
3. The absence of a claim does not imply insecurity.
4. Documentation reflects the system state at the time of publication.
5. Operational secrets, infrastructure topology, and private keys are not published.

Official documentation is published only via:
- https://atoposlabs.com
- https://github.com/wicki2c/atopos-security

If you discover a vulnerability, please follow DISCLOSURE_POLICY.md.
